Admission Portal

Project information

  • Project Name :Admission Portal
  • Role: Team Member
  • Technology Used: ReactJS, FastAPI
  • Team Size: 4

The aim of the project is to generate the gat hall ticket,gat call letter and sending email to students for the MSIT admission process. We have developed backend code that connects the database and stores the values into the database and extracts the dynamic data into hall ticket, call letter pdf. Then collect the email ids from the database and the admin can send the hall ticket to students to appear for the examination, gat call letters will be sent to the respective candidate for the MSIT counselling process.
